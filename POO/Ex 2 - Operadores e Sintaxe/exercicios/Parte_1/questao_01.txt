Sobre tipos primitivos na linguagem Java, responda o que se pede:
a. O que são tipos primitivos?
b. Qual a diferença entre tipos primitivos e objetos?
c. Pesquise e explique a diferença entre o tipo primitivo int e a classe Integer
d. Porque os tipos primitivos são mantidos na linguagem Java, uma vez que existem objetos
que podem realizar este papel?

Respostas:
a) São os tipos de dados que uma variável pode receber. Se for inteiro, então o dado é do tipo int, caso seja string o dado é do tipo char, etc.

b) Tipos primitivos se referem ao tipo de dado daquela variável, objeto são instâncias de uma classe que podem conter dados e métodos relacionados.

c) int é um tipo primitivo, representando números inteiros.
    Integer é uma classe de invólucro que encapsula um valor int em um objeto.
